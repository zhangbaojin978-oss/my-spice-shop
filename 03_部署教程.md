# 网店部署教程（GitHub Pages 保姆级）

> 将你的本地网店变成所有人都能访问的链接
> 部署方式：GitHub Pages（完全免费）

---

## 为什么选择 GitHub Pages？

| 优势 | 说明 |
|-----|------|
| 💰 完全免费 | 永久免费使用 |
| 🔗 链接稳定 | 链接永久有效 |
| 📱 支持手机 | 所有设备都能访问 |
| ⚡ 速度快 | 全球CDN加速 |
| 🎨 支持自定义 | 可绑定自己的域名 |

---

## 部署步骤（5步搞定）

### 步骤1：注册 GitHub 账号

1. 打开：https://github.com/
2. 点击右上角「Sign up」
3. 填写邮箱、密码、用户名，按提示完成注册

> 💡 提示：用户名会出现在你的网站链接里，选一个好听易记的

---

### 步骤2：创建仓库

1. 登录后，点击右上角 `+` 图标 → 选择「New repository」
2. 填写信息：
   ```
   Repository name: my-spice-shop
   Public: ✅ 选中（必须公开才能用 Pages）
   Add a README file: ✅ 勾选
   ```
3. 点击「Create repository」

---

### 步骤3：上传文件

#### 方法一：网页上传（推荐新手）

1. 点击「Add file」→ 「Upload files」
2. 把以下文件拖进去：
   - `index.html`（你的网店首页）
   - `02_包装获客.md`（可选）
   - `01_市场运营秘书.md`（可选）
3. 滚动到底部，填写提交信息：
   ```
   Commit changes: 第一次上传网店
   ```
4. 点击「Commit changes」

#### 方法二：git 命令上传（适合懂git的人）

```bash
cd D:\my-spice-shop
git init
git add .
git commit -m "第一次上传"
git branch -M main
git remote add origin https://github.com/你的用户名/my-spice-shop.git
git push -u origin main
```

---

### 步骤4：开启 GitHub Pages

1. 回到仓库首页
2. 点击最上方的 **Settings**（齿轮图标）
3. 左侧菜单找到 **Pages**
4. 在「Build and deployment」下设置：
   ```
   Source: Deploy from a branch
   Branch: main / (root)
   ```
5. 点击「Save」

---

### 步骤5：获取链接

1. 等待1-2分钟，刷新 Pages 页面
2. 你会看到链接：
   ```
   https://你的用户名.github.io/my-spice-shop/
   ```
3. 复制这个链接，在任何设备都能打开！

## 如何更新网站？

当你修改了 `index.html` 后，重新上传即可：

1. 回到仓库首页
2. 点击 `index.html`
3. 点击右上角「...」→「Delete file」删除旧文件
4. 点击「Add file」→「Upload files」上传新文件
5. 提交，等待1-2分钟自动更新

---

## 常见问题

**Q: 我改了链接上的文件，为什么还没变化？**
A: GitHub Pages 更新需要1-2分钟，等一下刷新

**Q: 我能看到，但朋友看不到？**
A: 确保你的仓库是 Public（公开的），Private 仓库无法用 Pages

**Q: 我能用自己的域名吗？**
A: 可以！在 Pages 设置里添加自定义域名，需要先买域名

**Q: 部署失败怎么办？**
A: 检查文件名是否是 `index.html`（大小写要一致）

## 进阶：绑定自己的域名（可选）

如果你买了域名（比如 `yishaoshihua.com`），可以这样绑定：

1. 买好域名后，在 Pages 设置里添加自定义域名
2. 按提示配置DNS解析
3. 等待DNS生效（通常几小时到1天）
